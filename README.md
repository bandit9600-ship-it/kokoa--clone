# KokoaTalk Clone

7-page KakaoTalk-style clone for GitHub Pages.

## Pages
- `index.html` login
- `friends.html` friends
- `chats.html` chats
- `chat.html` chat detail
- `find.html` find
- `more.html` more
- `settings.html` settings

## Deploy
1. Create a GitHub repository.
2. Upload all files in this folder.
3. Go to **Settings > Pages**.
4. Source: **Deploy from a branch**.
5. Branch: **main**, folder: **/root**.
6. Submit the `https://USERNAME.github.io/REPOSITORY/` link.
